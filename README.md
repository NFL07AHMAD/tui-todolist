# TUI Todo List

> **Note:** This application is still in early access. If you encounter any issues or bugs, feel free to open an issue. 😄

## 🤔 What is this?

A simple and lightweight terminal-based todo list application written in Java.

## 💻 Supported Platforms

This application runs on:

- Windows
- Linux
- macOS
- Android (requires Termux)

## Usage

### Installing and running

Make sure you have the latest version of Java installed.

Download the latest release, then run:

```bash
java -jar tui-todolist.jar
```

### Keybinds

` a `: Toggle complete task
` Tab `: Switch focus
